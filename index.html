    b) perguntas e respostas do item 2
  1. Na tag com atributo type="text", para que serve o atributo "placeholder"?

    2. Resposta: O atributo “placeholder” é usado para fornecer um texto de exemplo ou uma dica dentro do campo de entrada. Ele aparece como um texto cinza claro antes que o usuário insira qualquer valor.
  2. Na tag <'input'> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. Resposta: O atributo “checked” é usado para pré-selecionar um botão de rádio ou uma caixa de seleção. Quando definido como “checked”, o elemento estará marcado por padrão.
  3. Na tag <'input'> para que serve o atributo "value"?

    4. Resposta: O atributo “value” define o valor associado ao campo de entrada. Esse valor é enviado ao servidor quando o formulário é enviado.
  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. Resposta: Para agrupar vários botões de rádio de modo que apenas um deles fique marcado de cada vez, use o mesmo atributo “name” para todos os botões do grupo.
